### four sum

- **Definition:** To return the quadruplets whose sum reach the target 
- **Approach:** Two pinters approach
- **Idea:**
  - We initialize i and j and we use two pointer approach in the variables r and l 
  - then if the sum matches the target store in the arraylist and update the pointers , else directly update the pointers
  - at last check the edge test cases.
